# VoiceTyper v1.3

Десктопное приложение для голосового ввода текста. Распознаёт речь через **Google Gemini** или **Groq Whisper** и автоматически вставляет текст в любое активное окно.

Работает в system tray, не мешает и всегда под рукой.

## Возможности

- **Два провайдера на выбор** — Google Gemini или Groq Whisper, переключение в один клик
- **Push-to-Talk** — зажми хоткей, говори, отпусти — текст вставится в активное окно
- **Voice Activated** — автоматически определяет голос и вставляет текст при паузе
- **Любой язык речи** — провайдер сам определяет язык, не нужно переключать
- **Настраиваемый хоткей** — любая клавиша или комбинация (F8, Ctrl+F8, Alt+Shift+A и т.д.)
- **Выбор модели** — у каждого провайдера свой список моделей
- **Автоматический fallback** — при ошибке 429/500 мгновенно переключается на другую модель
- **Сохранение буфера обмена** — clipboard восстанавливается после вставки транскрипции
- **История транскрипций** — до 500 записей с возможностью копирования
- **3 языка интерфейса** — English, Русский, Українська
- **Автозапуск с Windows**
- **System tray** с цветовой индикацией состояния

## Быстрый старт

### Готовый exe (без Python)

1. Скачай `dist/VoiceTyper.exe` из этого репозитория
2. Запусти — появится окно настроек
3. Выбери провайдера (Gemini или Groq) и вставь свой API ключ
4. Нажми F8 (или свой хоткей) и говори

### Сборка из исходников

```bash
pip install -r requirements.txt
build.bat
```

Готовый exe появится в `dist/VoiceTyper.exe`.

## Провайдеры

### Google Gemini

- **Где взять ключ:** [Google AI Studio](https://aistudio.google.com/apikey) → Create API Key. Ключ начинается с `AIza...`
- **Бесплатный лимит:** 10 RPM, 250 запросов/день — для голосового ввода обычно достаточно
- **Модели:** `gemini-3-flash-preview`, `gemini-2.5-flash` (по умолчанию), `gemini-2.5-flash-lite`
- **Платный тариф:** [Paid Tier](https://ai.google.dev/gemini-api/docs/billing), ~$0.40-1.00 за час аудио

### Groq Whisper (рекомендуется)

Groq хостит опенсорс-модель Whisper от OpenAI на своих LPU-чипах — быстрее и дешевле Gemini.

- **Где взять ключ:**
  1. Зайди на [console.groq.com](https://console.groq.com) и зарегистрируйся (Google/GitHub OAuth, **банковская карта не нужна**)
  2. Слева: **API Keys** → **Create API Key** → скопируй ключ (начинается с `gsk_...`)
  3. Вставь в настройки VoiceTyper, в поле API key, при выбранном провайдере Groq
- **Бесплатный лимит:** 20 RPM, **2000 запросов в день**, **8 часов аудио в день**. Для активного ежедневного использования платить ничего не нужно.
- **Модели:**
  - `whisper-large-v3-turbo` — быстрая, $0.04/час сверх лимита (по умолчанию)
  - `whisper-large-v3` — максимальное качество, $0.111/час сверх лимита

## Настройки

| Параметр | Описание |
|----------|----------|
| Provider | Gemini или Groq |
| API Key | Ключ активного провайдера (хранится отдельно для каждого) |
| Hotkey | Хоткей для Push-to-Talk (по умолчанию F8) |
| Model | Модель провайдера для транскрипции |
| Language | Язык интерфейса (язык речи определяется автоматически) |
| Mode | Push-to-Talk или Voice Activated |
| Start with Windows | Автозапуск при загрузке системы |

Настройки сохраняются автоматически в `dist/settings.json`. Ключи Gemini и Groq хранятся независимо — можно переключаться между провайдерами без перенабора.

## Индикация в трее

| Цвет | Состояние |
|------|-----------|
| Синий | Готов — можно говорить |
| Красный | Идёт запись |
| Оранжевый | Обработка / ожидание / fallback |

## Что нового в v1.3

- **Groq Whisper как второй провайдер** — переключение Gemini ↔ Groq в настройках
- **Whisper Large v3 Turbo** по умолчанию для Groq — быстро и почти бесплатно (8 часов аудио/день на free tier)
- **Независимое хранение ключей** — Gemini и Groq ключи живут параллельно, переключение провайдера не сбрасывает второй ключ
- **Автоматическая миграция** старого `settings.json` (поля `api_key`/`model` → `gemini_api_key`/`gemini_model`)

## Что было в v1.2

- **Убран cooldown между записями** — запись доступна сразу после транскрипции, без задержки 6 секунд
- **Мгновенный повтор** — можно зажать хоткей сразу после вставки текста (в PTT и VAD режимах)
- **Индикация в трее** — иконка сразу переходит в "готов" (синяя) после вставки, без оранжевого ожидания

## Что было в v1.1

- **Автоматический fallback моделей** — при 429/500 мгновенно пробует следующую модель, без длительных retry
- **Сохранение буфера обмена** — clipboard восстанавливается после вставки (Win32 API с retry + pyperclip fallback)
- **Обновлённый system prompt** — явное требование Cyrillic для русского/украинского (исправлен транслит на flash-lite)
- **Обновлённый список моделей** — gemini-3-flash-preview, gemini-2.5-flash, gemini-2.5-flash-lite
- Удалены deprecated модели 2.0
